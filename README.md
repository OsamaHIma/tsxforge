# TsxForge 🛠️

**Powerful wrapper components extending generic UI primitives with declarative APIs and robust integrations.**

TsxForge is a collection of high-performance, accessible, and highly customizable React components built on top of Tailwind CSS and shadcn/ui. It is designed to reduce boilerplate and provide a premium developer experience for building modern web applications.

## ✨ Features

- 🧩 **Advanced Wrappers**: Components like `Table`, `Input`, and `DateTimePicker` with built-in logic.
- 🎨 **Modern Design**: Premium aesthetics with smooth animations and curated color palettes.
- ♿ **Accessible**: Built with Radix UI primitives ensuring ARIA compliance.
- 🚀 **Performance**: Optimized for speed, including features like lazy loading for images.
- 📚 **Documentation**: Comprehensive documentation powered by Fumadocs.
- 🛠️ **Developer Friendly**: Strong TypeScript support and declarative APIs.

## 📦 Included Components

### Form Elements
- **Input & Textarea**: Floating labels, animated states, and built-in icons.
- **Phone Input**: International support with country flags.
- **Uploaders**: Specialized components for Images and Documents with drag-and-drop.
- **Color & DateTime Pickers**: Robust popover-based selection tools.

### Action & Feedback
- **Button & ButtonLink**: Loading states, content slots, and TanStack Router integration.
- **Action Confirmation**: Declarative modals for critical user actions.

### Data Display
- **Table**: Feature-rich tables with horizontal scrolling and card-style rendering.
- **Pagination**: Logic-driven navigation for large datasets.
- **Empty State**: Ready-to-use layouts for missing data or loading skeletons.
- **Image**: Lazy-loading image component with blur-up effects.

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- pnpm (recommended)

### Installation

```bash
pnpm install
```

### Development

Run the development server:

```bash
pnpm dev
```

### Build

Create a production-ready build:

```bash
pnpm build
```
