## Installation

The easiest way to add TsxForge components to your project is via the CLI. This will automatically install the component in your `components/tsxforge` directory and handle all required Shadcn UI dependencies and npm packages.

```bash
npx shadcn@latest add https://tsxforge.vercel.app/registry/[component].json
```

Replace `[component]` with the name of the component you want to add (e.g., `input`, `button`, `datetime-picker`).
