# Textarea

The `Textarea` component resembles the `Input` component but is tailored for multiline text entries. It includes the same quality-of-life enhancements for labels, structural nesting, and loading states.

## Custom Props

| Prop            | Type              | Default     | Description                                                                                            |
| --------------- | ----------------- | ----------- | ------------------------------------------------------------------------------------------------------ |
| `label`         | `string`          | `undefined` | The label text.                                                                                        |
| `animatedLabel` | `boolean`         | `true`      | Enables floating label animation inside the textarea wrapper.                                          |
| `startContent`  | `React.ReactNode` | `undefined` | Icon or element displayed alongside the text field.                                                    |
| `endContent`    | `React.ReactNode` | `undefined` | Icon or element displayed at the end.                                                                  |
| `isLoading`     | `boolean`         | `false`     | Displays a loading spinner inside the textarea wrapper.                                                |
| `classNames`    | `object`          | `{}`        | Target specific sub-elements: `wrapper`, `main`, `startContent`, `endContent`, `description`, `label`. |
