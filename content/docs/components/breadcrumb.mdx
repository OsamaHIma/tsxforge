# Breadcrumb

The `Breadcrumb` component provides a simplified, declarative way to render breadcrumb navigation. Unlike the standard Shadcn UI breadcrumb which requires manual orchestration of lists and items, the TsxForge version automatically handles structure and separators.

## Why TsxForge Breadcrumb?

The primary advantage is **conciseness**. The TsxForge Breadcrumb automatically wraps its children in logic that inserts separators and manages active/inactive states based on position.

### Comparison

**Standard Shadcn UI (Verbose):**

```tsx
<Breadcrumb>
  <BreadcrumbList>
    <BreadcrumbItem>
      <BreadcrumbLink href="/">Home</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbLink href="/components">Components</BreadcrumbLink>
    </BreadcrumbItem>
    <BreadcrumbSeparator />
    <BreadcrumbItem>
      <BreadcrumbPage>Breadcrumb</BreadcrumbPage>
    </BreadcrumbItem>
  </BreadcrumbList>
</Breadcrumb>
```

**TsxForge (Declarative):**

```tsx
<Breadcrumb>
  <BreadcrumbLink to="/">Home</BreadcrumbLink>
  <BreadcrumbLink to="/components">Components</BreadcrumbLink>
  <BreadcrumbPage>Breadcrumb</BreadcrumbPage>
</Breadcrumb>
```
