# ColorPicker

The `ColorPicker` component allows users to select a color from a Chrome-style picker. It is built using a Popover and integrates seamlessly with the design system.

## Features

- **Popover Integration**: Saves screen real estate by opening the picker in a toggleable overlay.
- **ChromePicker**: Familiar interface for precise color selection.
- **Accessibility**: Includes optional label support with `htmlFor` matching.

## Custom Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `color` | `string` | `"#000000"` | The current hex color value. |
| `onChange` | `(color: string) => void` | `undefined` | Callback triggered when a new color is selected. |
| `label` | `string` | `undefined` | Optional label text displayed above the picker button. |
| `id` | `string` | `undefined` | Unique identifier for the component. |
| `className` | `string` | `undefined` | Additional CSS classes for the container. |
