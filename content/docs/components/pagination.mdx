# Pagination

The `Pagination` component eliminates the boilerplate associated with manual pagination construction. Provide the current state, and the component calculates the correct ellipsis placements and renders the buttons.

## Usage

The most basic implementation requires managing the current page state and providing the total number of pages.

<Tabs items={['Preview', 'Code']}>
  <Tab value="Preview">
    <div className="not-prose flex flex-col items-center gap-6 p-4">
      <PaginationBasicDemo />
    </div>
  </Tab>
  <Tab value="Code">
```tsx
import Pagination from "@/components/tsxforge/pagination"
import { useState } from "react"

export default function Example() {
  const [page, setPage] = useState(1);
  const totalPages = 10;

  return (
    <div className="flex flex-col items-center gap-6 p-4 border rounded-xl bg-muted/20">
      <Pagination 
        currentPage={page} 
        totalPages={totalPages} 
        onChange={setPage} 
      />
    </div>
  )
}
```

## Items to Display

You can customize how many page buttons are visible before the ellipsis appears using the `paginationItemsToDisplay` prop.

  <Tab value="Preview">
    <div className="not-prose flex flex-col items-center gap-6 p-4">

    </div>
  </Tab>
  <Tab value="Code">
```tsx
import Pagination from "@/components/tsxforge/pagination"
import { useState } from "react"

export default function ConfigurableExample() {
  const [page, setPage] = useState(1);

  return (
    <Pagination 
      currentPage={page} 
      totalPages={20} 
      paginationItemsToDisplay={5}
      onChange={setPage} 
    />
  )
}
```

## Custom Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `currentPage` | `number` | **Required** | The currently selected page number (1-indexed). |
| `totalPages` | `number` | **Required** | The maximum number of pages available. |
| `paginationItemsToDisplay` | `number` | `5` | The maximum number of individual page buttons to display before inserting an ellipsis `...`. |
| `onChange` | `(page: number) => void` | `undefined` | Callback invoked when a user clicks a page button or the next/prev arrows. |

*Note: If `totalPages` is 1 or falsey, the component will return `null` and not render itself.*
