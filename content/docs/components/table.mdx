# Table

A set of components that allow for a highly customizable yet standard way to display data in a tabular format. It includes built-in horizontal scrolling on smaller viewports.

## Components

### Table
The root container that includes a horizontal `ScrollArea`.

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `classNames` | `object` | `undefined` | `{ wrapper?: string }` to customize the ScrollArea. |
| `className` | `string` | `undefined` | CSS class for the `table` element. |

### TableHeader
Wrapper for header rows (`thead`).

### TableBody
Wrapper for data rows (`tbody`).

### TableRow
A row in the table (`tr`).

### TableHead
A header cell (`th`).

### TableCell
A standard cell (`td`).

### TableFooter
Wrapper for footer content (`tfoot`).

### TableCaption
A caption for the table content.
