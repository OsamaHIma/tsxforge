# Image

The `Image` component is an enhanced wrapper for `react-lazy-load-image-component`. It abstracts away lazy loading behaviors, provides optional blurred effects while the image is fetching, and manages error handling intelligently with fallback sources.

## Features

- **Lazy Loading**: Automatically uses `loading="lazy"` off-screen to improve page performance.
- **Blur Effects**: Images can start as a blur effect while downloading, fading clearly to the actual image.
- **Error Fallbacks**: Pre-configured behavior swaps a broken source for `fallbackSrc` preventing broken image UI states.

## Custom Props

This component also automatically accepts the props of `react-lazy-load-image-component`.

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `src` | `string` | **Required** | The source URL of the image. |
| `alt` | `string` | **Required** | Accessible description of the image content. |
| `fallbackSrc` | `string` | `'/image-break.png'` | Source URL of the image replacement if `src` fails to load. |
| `showBlurEffect` | `boolean` | `true` | If true, applies a CSS blur and fades in opacity the final image string. |
| `placeholder` | `React.ReactNode` | `undefined` | Custom React element used as a loading skeleton placeholder. |
| `onImageLoad` | `() => void` | `undefined` | Callback fired successfully after an image completely initializes. |
| `onImageError` | `(error) => void` | `undefined` | Further custom fallback behavior. |
