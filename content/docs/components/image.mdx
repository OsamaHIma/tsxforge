# Image

The `Image` component is built on top of `react-lazy-load-image-component`. It provides a simple API for lazy loading, blur-up loading effects, and customizable fallback images for error handling.

## Features

- **Lazy Loading**: Automatically defers loading of images until they enter the viewport.
- **Blur Effect**: Displays a smooth blur-up transition as the image loads.
- **Error Handling**: Gracefully falls back to a placeholder image when loading fails.
- **Placeholders**: Supports custom loading placeholder components.

## Custom Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `src` | `string` | `undefined` | The source URL of the image. |
| `alt` | `string` | `undefined` | Alt text for accessibility. |
| `fallbackSrc` | `string` | `"/image-break.png"` | Image URL to use if the main `src` fails. |
| `showBlurEffect` | `boolean` | `true` | Whether to show a blur effect during loading. |
| `placeholder` | `React.ReactNode` | `undefined` | Custom element to show while loading. |
| `onImageLoad` | `() => void` | `undefined` | Callback fired when the image successfully loads. |
| `onImageError` | `(error: Event) => void` | `undefined` | Callback fired if image loading fails. |
| `className` | `string` | `undefined` | Additional CSS classes for the image element. |

> [!TIP]
> This component extends `LazyLoadImageProps`, so you can use all properties from the original `react-lazy-load-image-component` such as `scrollPosition`, `threshold`, and `visibleByDefault`.
