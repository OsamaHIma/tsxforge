# Image

The `Image` component is built on top of `react-lazy-load-image-component`. It provides a simple API for lazy loading, blur-up loading effects, and customizable fallback images for error handling.

## Blur Effect

The `showBlurEffect` prop enables a smooth blur-up transition as the image loads. This is useful for providing immediate visual feedback especially on slower connections.

<Tabs items={['Preview', 'Code']}>
  <Tab value="Preview">
    <div className="not-prose p-2 w-full overflow-hidden">
      <ImageBlurDemo />
    </div>
  </Tab>
  <Tab value="Code">
```tsx
import { useState } from 'react'
import { Image, Button } from "@/components/tsxforge"

export default function BlurExample() {
  const [key, setKey] = useState(0)

  return (
    <div className="flex flex-col gap-4">
      <Image
        key={key}
        src={`https://images.unsplash.com/photo-1542291026-7eec264c27ff?q=80&w=2070&auto=format&fit=crop&t=${key}`}
        alt="Red Nike Shoe"
        showBlurEffect={true}
        className="rounded-xl overflow-hidden shadow-lg"
      />
      <Button onClick={() => setKey(k => k + 1)}>
        Re-trigger Blur
      </Button>
    </div>
  )
}
```

## Error Handling

Gracefully handle broken image links by providing a `fallbackSrc`. The component automatically switches to this source if the primary `src` fails to load.

<Tabs items={['Preview', 'Code']}>
  <Tab value="Preview">
    <div className="not-prose p-2 w-full overflow-hidden">
      <ImageErrorDemo />
    </div>
  </Tab>
  <Tab value="Code">
```tsx
import { Image } from "@/components/tsxforge/image"

export default function ErrorExample() {
  return (
    <Image
      src="https://broken-link.com/image.jpg"
      fallbackSrc="/image-break.png"
      alt="Broken Image Example"
      className="rounded-xl shadow-lg"
    />
  )
}
```

## Features

- **Lazy Loading**: Automatically defers loading of images until they enter the viewport.
- **Blur Effect**: Displays a smooth blur-up transition as the image loads.
- **Error Handling**: Gracefully falls back to a placeholder image when loading fails.
- **Placeholders**: Supports custom loading placeholder components.

## Custom Props

| Prop             | Type                     | Default              | Description                                       |
| ---------------- | ------------------------ | -------------------- | ------------------------------------------------- |
| `src`            | `string`                 | `undefined`          | The source URL of the image.                      |
| `alt`            | `string`                 | `undefined`          | Alt text for accessibility.                       |
| `fallbackSrc`    | `string`                 | `"/image-break.png"` | Image URL to use if the main `src` fails.         |
| `showBlurEffect` | `boolean`                | `true`               | Whether to show a blur effect during loading.     |
| `placeholder`    | `React.ReactNode`        | `undefined`          | Custom element to show while loading.             |
| `onImageLoad`    | `() => void`             | `undefined`          | Callback fired when the image successfully loads. |
| `onImageError`   | `(error: Event) => void` | `undefined`          | Callback fired if image loading fails.            |
| `className`      | `string`                 | `undefined`          | Additional CSS classes for the image element.     |

> [!TIP]
> This component extends `LazyLoadImageProps`, so you can use all properties from the original `react-lazy-load-image-component` such as `scrollPosition`, `threshold`, and `visibleByDefault`.
