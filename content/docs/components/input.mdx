# Input

The `Input` component creates a robust text field that handles its own label, state animations, leading/trailing icons, and password visibility toggling, significantly reducing boilerplate code for forms.

### Password Input

Setting the `type` to `"password"` automatically provides a toggle button to reveal or outline the text.

```tsx
<Input label="Password" type="password" showPasswordToggle={true} />
```

## Custom Props

| Prop                 | Type              | Default     | Description                                                                                            |
| -------------------- | ----------------- | ----------- | ------------------------------------------------------------------------------------------------------ |
| `label`              | `string`          | `undefined` | The label text.                                                                                        |
| `animatedLabel`      | `boolean`         | `false`     | Enables floating label animation inside the input wrapper.                                             |
| `startContent`       | `React.ReactNode` | `undefined` | Icon or element preceding the input value.                                                             |
| `endContent`         | `React.ReactNode` | `undefined` | Icon or element following the input value.                                                             |
| `showContentDivider` | `boolean`         | `true`      | Show a vertical divider line after the `startContent`.                                                 |
| `showPasswordToggle` | `boolean`         | `true`      | Show a built-in eye icon to toggle password visibility (only active if `type="password"`).             |
| `isLoading`          | `boolean`         | `false`     | Displays a loading spinner at the end of the input field.                                              |
| `classNames`         | `object`          | `{}`        | Target specific sub-elements: `wrapper`, `main`, `startContent`, `endContent`, `description`, `label`. |
