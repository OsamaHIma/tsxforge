# DateTimePicker

The `DateTimePicker` component allows users to select a date and time with high precision. It supports 12/24 hour cycles, multiple granularity levels, and localized formatting.

## Installation

## Features

- **Precise Timing**: Selection for hours, minutes, and seconds.
- **Hour Cycles**: Easy switching between 12-hour (AM/PM) and 24-hour formats.
- **Granularity Control**: Focus on the level of detail needed (day, hour, minute, or second).
- **Date Range**: Customizable year range for quick navigation.
- **Formatting**: Integrated `date-fns` formatting for display and output.

## Custom Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `value` | `Date` | `undefined` | The selected date and time. |
| `onChange` | `(date: Date \| undefined) => void` | `undefined` | Callback for date selection. |
| `onChangeFormatted` | `(formatted: string \| undefined) => void` | `undefined` | Callback returning formatted date string. |
| `disabled` | `boolean` | `false` | Whether the picker is disabled. |
| `hourCycle` | `12 \| 24` | `12` | The time format (AM/PM or 24h). |
| `placeholder` | `string` | `"Pick a date"` | Toggle button placeholder text. |
| `yearRange` | `number` | `50` | Range of years to display in the dropdown. |
| `granularity` | `day \| hour \| minute \| second` | `"second"` | Smallest unit displayed. |
| `displayFormat` | `object` | `{ hour12: "dd-MM-yyyy" }` | Format string for the toggle label. |
| `outputFormat` | `string` | `"dd-MM-yyyy"` | Format string for `onChangeFormatted`. |
| `locale` | `Locale` | `enUS` / `ar` | `date-fns` locale for localization. |
