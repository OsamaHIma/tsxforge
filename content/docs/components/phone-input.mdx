# Phone Input

The `PhoneInput` component leverages `react-phone-number-input` to provide a seamless interface for capturing international phone numbers securely, along with a searchable dropdown list of countries.

## Features

- **Built-in Validation**: Utilizes `libphonenumber-js` under the hood.
- **Searchable Country Code**: Rendered via a customized Popover to search through countries and their dial codes easily.
- **Auto-Formatting**: Automatically formats the phone string as the user inputs based on the selected country format.

## Custom Props

This component passes through all the standard `react-phone-number-input` props, as well as native input attributes.

| Prop        | Type                     | Default     | Description                                                                       |
| ----------- | ------------------------ | ----------- | --------------------------------------------------------------------------------- |
| `value`     | `string`                 | `undefined` | The normalized E.164 string format.                                               |
| `onChange`  | `(value: Value) => void` | `undefined` | A callback returning the parsed phone number string.                              |
| `disabled`  | `boolean`                | `false`     | Sets the entire field (both the text input and the country selector) to disabled. |
| `className` | `string`                 | `undefined` | Class names passed to the outermost wrapper.                                      |
